Sobre a Oficina

Esta oficina oferece uma introdução prática e acessível ao mundo da Machine Learning e Ciência de Dados. Partindo do zero, os alunos aprenderão os conceitos fundamentais através de exemplos concretos e hands-on, desenvolvendo gradualmente as habilidades necessárias para construir seu primeiro modelo preditivo.

O que você vai aprender:


Fundamentos: O que é Machine Learning, onde se aplica no mundo real e como funciona um projeto de dados do início ao fim.
Ferramentas: Trabalharemos com Python, Pandas e Scikit-learn no Google Colab - tudo gratuito e na nuvem.
Fluxo de Dados: Desde a importação e exploração de dados até a construção e avaliação de modelos.
Pré-processamento: Como limpar e preparar dados para análise.
Modelos Básicos: Introdução aos algoritmos de classificação.

Projeto Final - Sobrevivência no Titanic:


Você vai analisar os dados históricos do Titanic e construir um modelo que prevê quais passageiros sobreviveram ao naufrágio. Este projeto emblemático da comunidade de Data Science permitirá aplicar todo o conhecimento adquirido:
Analisar quais fatores influenciavam a sobrevivência (classe, gênero, idade).
Limpar e preparar os dados.
Treinar um modelo de Machine Learning.
Avaliar o desempenho do seu modelo.

Para quem é este curso:

Alunos sem experiência prévia em Machine Learning.
Quem tem conhecimento básico de programação (ou nenhum).
Interessados em entender na prática como a ciência de dados funciona.
